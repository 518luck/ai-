# 概念定义：Agent / Command / Skill

| 概念 | 本质 | 作用位置 | 类比 |
|------|------|----------|------|
| **Agent** | 角色配置 + 系统提示词 + 权限 + 模型 | 决定"谁来工作" | 招聘了一位专家 |
| **Command** | 一次性任务模板 | 决定"现在要做什么" | 下达了一个任务指令 |
| **Skill** | 可按需加载的知识/规则包 | 决定"做这类事时应该遵守什么规则" | 发了一本操作手册 |

## 它们的关系

```
Command（任务指令）
  └→ Agent（执行者）
       └→ Skill（参考规则）
            └→ AGENTS.md（项目基础规范）
```

一次典型的 AI 编程交互流程：

1. 用户输入一个 Command（如 `/commit`）
2. 系统根据 Command 配置派遣对应的 Agent
3. Agent 加载其关联的 Skills 获取专业知识
4. 所有操作都受 AGENTS.md 中定义的项目级规范约束
