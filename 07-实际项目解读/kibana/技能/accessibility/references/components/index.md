# EUI 组件指南 — 标准的无障碍用法 (EUI component guides — canonical accessible usage)

打开与您正在编写或重构的组件相对应的指南。请先阅读 **`../shared_principles.md`**；以下每个指南都是对这些通用原则的延伸与细化。

| 指南 (Guide) | 主题 (Topic) | EUI 组件 (EUI components) |
|---|---|---|
| `callouts.md` | 提示框（在条件触发时进行播报） | `EuiCallOut` |
| `data_tables.md` | 数据表格（表格标题/caption） | `EuiBasicTable`、`EuiInMemoryTable` |
| `focus_and_keyboard.md` | 焦点与键盘交互（Tab 键顺序、工具提示锚点） | `EuiButton`、`EuiButtonIcon`、`EuiLink`、`EuiToolTip` |
| `form_layout.md` | 表单行与无效状态（非合法校验状态） | `EuiFormRow`、`EuiFieldText`、`EuiFieldNumber`、`EuiFilePicker`、`EuiComboBox`、`EuiTextArea`、`EuiSelect` |
| `icons_and_tooltips.md` | 图标与图标提示（装饰性图标 vs 具有实际含义的图标） | `EuiIcon`、`EuiIconTip`、`EuiToolTip` |
| `interactive_components.md` | 交互式控件的无障碍命名 | `EuiBetaBadge`、`EuiButtonIcon`、`EuiComboBox`、`EuiSelect`、`EuiSuperSelect`、`EuiPagination`、`EuiTreeView`、`EuiBreadcrumbs` |
| `overlays.md` | 覆盖物层（模态框、侧滑抽屉、气泡卡片） | `EuiModal`、`EuiFlyout`、`EuiFlyoutResizable`、`EuiConfirmModal`、`EuiPopover` |
| `radio_groups.md` | 单选框组（通过 `name` 属性分组） | `EuiRadio`、`EuiRadioGroup` |
| `tooltip_icon.md` | 图标按钮上的工具提示（避免读屏软件 SR 重复播报文本） | `EuiToolTip`、`EuiButtonIcon` |
