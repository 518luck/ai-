# SST v3 Monorepo 项目

这是一个使用 TypeScript 开发的 SST v3 monorepo 项目。该项目使用 bun workspaces 进行包管理。

## 项目结构 (Project Structure)

- `packages/` - 包含所有工作区包（包含 functions、core、web 等）
- `infra/` - 按服务拆分的基础设施定义文件（如 storage.ts、api.ts、web.ts）
- `sst.config.ts` - 采用动态导入的主 SST 配置文件

## 代码标准 (Code Standards)

- 使用 TypeScript 并启用严格模式（strict mode）
- 共享代码应当放入 `packages/core/` 中，并配置正确的导出（exports）
- 函数代码应当放入 `packages/functions/`
- 基础设施定义应当拆分为逻辑文件存放在 `infra/` 中

## Monorepo 约定 (Monorepo Conventions)

- 使用工作区名称来导入共享模块：`@my-app/core/example`
